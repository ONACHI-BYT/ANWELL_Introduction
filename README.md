# ANWELL by BYT — Bản mẫu MVP

Ba bản mẫu giao diện tương tác của nền tảng chăm sóc sức khoẻ ANWELL.

| Trang | File | Dành cho |
|---|---|---|
| Ứng dụng cho người dùng | `ANWELL_MVP_KhachHang.html` | Khách đặt lịch trị liệu |
| Cổng thông tin & đăng ký | `ANWELL_CongDoiTac.html` | Đối tác nộp hồ sơ tham gia |
| Công cụ Quản trị | `ANWELL_Portal_QuanTri.html` | ADMIN, Trung tâm, KTV, Đối tác Kinh doanh, Cơ sở Đào tạo |

`index.html` là trang chủ liên kết tới cả ba.

## Cách đưa lên GitHub Pages

1. Tạo repository mới trên GitHub.
2. Tải lên cả **4 file HTML** trong thư mục này, giữ nguyên tên file.
3. Vào **Settings → Pages**, mục *Source* chọn nhánh `main` và thư mục `/ (root)`, bấm **Save**.
4. Sau vài phút, trang sẽ chạy tại `https://<tên-tài-khoản>.github.io/<tên-repo>/`

## Lưu ý

Đây là bản mẫu giao diện. Toàn bộ số liệu hiển thị là dữ liệu mô phỏng, chưa kết nối máy chủ hay cơ sở dữ liệu thật.

Cả ba bản mẫu đều có thanh chuyển kích thước màn hình (Siêu rộng · Laptop · Tablet · Điện thoại · Phone nhỏ) và chuyển giao diện (Thanh · Onsen) ở phía trên.
